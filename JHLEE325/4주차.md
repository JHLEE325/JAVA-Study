## 제어문
제어문은 프로그램의 실행흐름(Control Flow)를 결정하는 문법 <br>
CPU는 기본적으로 위에서 아래로 순차 실행하지만, 제어문을 통해 분기, 반복, 탈출을 제어

## 1. 선택문

선택문은 조건식의 평가 결과에 따라 서로 다른 실행 경로 중 하나를 선택하도록 제어 흐름을 변경하는 문장

<br>

### if 문

기본구조
```java
if (condition) {
    // 실행문
} else if (condition2) {
    // 실행문
} else {
    // 실행문
}
```
특징
- 조건식은 반드시 `boolean`
- C와 달리 `if(1)` 같은 문법 불가
- 중첩 가능
- 조건이 true인 경우 실행문 실행

<br>

### 삼항 연산자

기본구조
```java
int max = (a > b) ? a : b;
```
특징
- 표현식임
- 반환값이 존재
- if문보다 간결하지만 많이 중첩되는 경우 가독성이 떨어짐

<br>

### switch 문

기본구조
```java
switch (value) {
    case 1:
        break;
    case 2:
        break;
    default:
}
```
특징
- `byte`, `short`, `char`, `int`, `enum`, `String`
- `long`, `float`, `double` 불가
- fall-through 존재 = break문이 없으면 아래 적용되는 case도 전부 실행

#### 왜 long은 switch에 사용을 못하는가?
switch는 내부적으로
- `tableswitch`
- `lookupswitch`
바이트코드를 사용하는데 이는 int 기반 점프 테이블이라 64bit 매핑이 어려움

<br>

### JAVA 13+ switch 표현식

기본구조
```java
int result = switch (day) {
    case MONDAY -> 1;
    case TUESDAY -> 2;
    default -> 0;
};
```
특징
- fall-through 구조가 아니어서 break 필요가 없음
- 값 반환 가능
- `yield` 키워드 존재

#### 기존 switch의 문제점
- break 누락 버그 / 오류
- 값 할당 반복
- 가독성 저하

이를 해결하기 위한 표현식 형태로 개선한 내용

## 2. 반복문

특정 조건이 만족되는 동안 동일한 실행 블록을 반복 수행하도록 제어 흐름을 구성하는 문장

<br>

### for문

기본구조
```java
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}
```
실행순서
1. 초기식
2. 조건식
3. 실행
4. 증감식
5. 조건식 재평가

<br>

### 향상된 for문 (Enhanced for, for-each)

기본구조
```java
for (String s : list) {
    System.out.println(s);
}

// 배열의 경우 내부 작동 흐름
for (int i = 0; i < arr.length; i++)

// 컬렉션의 경우 내부 작동 흐름
Iterator it = list.iterator();
while(it.hasNext()) {
    String s = it.next();
}
```

<br>

### while문

기본구조
```java
while (condition) {
// 실행코드
}
```
- 선조건 반복
- 조건 먼저 검사 후 true인경우 실행코드 실행

<br>

### do-while문

기본구조
```java
do {
// 실행코드
} while(condition);
```
- 후조건 반복
- 최소 1번 실행 후 조건 검사
- 최소 1회 실행문 실행을 보장

### break / continue
break : 가장 가까운 반복문 탈출 <br>
continue : 현재 반복 스킵 후 다은 반복

label을 사용한 break
```java
outer:
for (...) {
    for (...) {
        break outer;
    }
}
// outer 가 break됨
```

## 3. 선택문 vs 반복문
| 구분     | 선택문                | 반복문          |
| ------ | ------------------ | ------------ |
| 목적     | 분기                 | 반복           |
| 내부 동작  | 조건 분기 점프           | 조건 + jump 반복 |
| JVM 관점 | branch instruction | loop jump    |
| 최적화    | 분기 예측              | 루프 최적화       |
| 주의점    | fall-through       | 무한루프         |
